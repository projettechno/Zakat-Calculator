Business Zakat Calculator

A local-first web portal for Moroccan businesses to calculate their annual zakat obligation. All data stays on the user's device — no accounts on a server, no data collection.

Features

Multi-business dashboard — manage zakat calculations for separate entities
Step-by-step wizard: Settings → Assets → Liabilities → Result
Auto-save — data persists in browser localStorage
Export/import business data as JSON for backups
Print-ready result page
Morocco-focused: currency fixed to MAD, nisab presets in dirhams
